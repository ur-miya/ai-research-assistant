# AI Research Assistant

AI-powered research assistant that helps find and analyze academic papers using RAG and LangGraph.

## Features
- Semantic search over arXiv papers
- Interactive Q&A about research papers
- Citation tracking and summarization

## Tech Stack
- Python 3.10+
- LangGraph for agent orchestration
- Qdrant as vector database
- Sentence Transformers for embeddings
- Docker for containerization
